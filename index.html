NFT visualization Project ->

On the right/bottom is another I am currently working on: it's a NFT price visualization project based on the data provided from moralis.

This project is front-end only. Where as I had used APIs provided from moralis to load, process, and analyze, various data about these non-fungible tokens.


In this "NFT broser", user can view the profile about various NFTs by making a simple search. Other infomation like name, description, and externalink were also included for user's convinence.


Finally and most importantly, historical prices fluctuation of the NFTs were included in this project.



Moralis for APIs.

Ant design for UI/UX.
